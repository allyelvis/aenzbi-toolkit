# **App Name**: Aenzbi Toolkit

## Core Features:

- Script Display: Display cross-platform scripts for Linux (Bash), Windows (PowerShell), and cloud integrations.
- Template Showcase: Show prebuilt automation templates for system setup, cloud provisioning, app deployment, CI/CD pipelines, and financial workflows.
- Secrets Management: Securely manage encrypted secrets using environment variables or vaults.
- Access Control: Enable role-based access to scripts and automation tools.
- Logging & Rollback: Generate comprehensive logs for auditing and rollback.
- Script Translation: AI Tool: Auto-generate cross-platform script translations from Bash to PowerShell and vice versa.
- Interactive Docs: Provide interactive documentation and usage examples for each script module.

## Style Guidelines:

- Primary color: Saturated blue (#29ABE2) to evoke a sense of technology, trust, and automation. 
- Background color: Light gray (#F0F0F0), a desaturated tint of the primary hue, providing a clean and modern backdrop that ensures legibility.
- Accent color: Yellow-orange (#FFB347), analogous to the primary hue, it provides contrast and draws attention to interactive elements.
- Body text and headline font: 'Inter', a grotesque-style sans-serif providing a modern and neutral look that's suitable for both.
- Code Font: 'Source Code Pro', a monospaced sans-serif, for clear display of code snippets.
- Use consistent and clear icons to represent different script categories (system, cloud, fin, dev).
- Clean, grid-based layout to organize modules, templates, and scripts effectively.
